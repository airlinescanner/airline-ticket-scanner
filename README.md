# Airline Ticket Scanner

Мобильное приложение на React Native (iOS + Android) для сканирования авиабилетов с экрана ноутбука с помощью камеры телефона.

## Технологический стек

- **Платформа**: React Native + TypeScript (Expo)
- **OCR**: Google ML Kit Text Recognition (on-device)
- **Камера**: react-native-vision-camera
- **База данных**: expo-sqlite (SQLite)
- **Навигация**: React Navigation (Bottom Tab + Stack)
- **Уведомления**: expo-notifications (локальные)
- **Локализация**: i18next + react-i18next
- **Тестирование**: Jest + fast-check (property-based testing)

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск на iOS
npm run ios

# Запуск на Android
npm run android

# Запуск в веб-браузере
npm run web
```

## 🔑 Обязательные переменные окружения

Для корректной работы распознавания (OCR) и искусственного интеллекта (AI) необходимо создать файл `.env` в корневой директории и заполнить следующие переменные:

### Основные ключи (Используют префикс `EXPO_PUBLIC_` для доступа в React Native):

* **`EXPO_PUBLIC_GROQ_API_KEY`** (Обязательный) — Ключ для Groq API. Используется как сверхбыстрый и дешевый LLM-парсер для извлечения структурированных данных из сырого текста OCR билета. Получить: [console.groq.com](https://console.groq.com/keys)
* **`EXPO_PUBLIC_MISTRAL_API_KEY`** (Обязательный) — Ключ для Mistral AI. Используется в качестве резервного парсера билетов, а также основного сервиса для извлечения правил регистрации авиакомпаний. Получить: [console.mistral.ai](https://console.mistral.ai/api-keys/)
* **`EXPO_PUBLIC_TAVILY_API_KEY`** (Обязательный) — Ключ для Tavily Search API. Применяется для "самолечения" неактивных веб-ссылок регистрации и умного онлайн-поиска правил авиакомпаний. Получить: [app.tavily.com](https://app.tavily.com/home)

### Опциональные настройки моделей:

* `EXPO_PUBLIC_MISTRAL_MODEL` — Модель для Mistral (по умолчанию `mistral-small-latest`, либо мультимодальная `pixtral-12b-2409`).
* `EXPO_PUBLIC_GROQ_MODELS` — Список моделей Groq через запятую для каскадного резервирования (например, `llama-3.3-70b-versatile,llama-3.2-90b-vision-preview`).

> [!WARNING]
> Если переменные окружения не настроены, приложение продолжит работать локально (включая локальный OCR и регулярный офлайн-парсер), но функции облачного ИИ-анализа и веб-обогащения будут недоступны.


## Основные возможности

- ✅ Сканирование авиабилетов через камеру
- ✅ OCR-распознавание текста (on-device, без интернета)
- ✅ Локальная база данных авиакомпаний (SQLite)
- ✅ Push-уведомления о начале регистрации на рейс
- ✅ Поддержка двух языков (украинский/русский)
- ✅ Светлая и тёмная темы оформления
- ✅ Резервное копирование и восстановление данных
