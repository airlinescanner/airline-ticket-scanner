# Airline Ticket Scanner

Мобильное приложение на React Native (iOS + Android) для сканирования авиабилетов с экрана ноутбука с помощью камеры телефона.

## Технологический стек

- **Платформа**: React Native + TypeScript (Expo)
- **OCR**: Google ML Kit Text Recognition (on-device)
- **Камера**: react-native-vision-camera
- **База данных**: expo-sqlite (SQLite)
- **Навигация**: React Navigation (Bottom Tab + Stack)
- **Уведомления**: expo-notifications (локальные)
- **Локализация**: i18next + react-i18next
- **Тестирование**: Jest + fast-check (property-based testing)

## Структура проекта

```
src/
├── components/          # Переиспользуемые UI-компоненты
├── screens/             # Экраны приложения
├── navigation/          # Настройка навигации
├── hooks/               # Кастомные хуки
├── services/            # Бизнес-логика и внешние интеграции
│   └── database/        # Работа с SQLite
├── theme/               # Дизайн-система (токены, ThemeContext)
├── i18n/                # Локализация (украинский, русский)
├── types/               # TypeScript типы и интерфейсы
└── utils/               # Вспомогательные функции
```

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск на iOS
npm run ios

# Запуск на Android
npm run android

# Запуск в веб-браузере
npm run web
```

## Основные возможности

- ✅ Сканирование авиабилетов через камеру
- ✅ OCR-распознавание текста (on-device, без интернета)
- ✅ Локальная база данных авиакомпаний (SQLite)
- ✅ Push-уведомления о начале регистрации на рейс
- ✅ Поддержка двух языков (украинский/русский)
- ✅ Светлая и тёмная темы оформления
- ✅ Резервное копирование и восстановление данных
