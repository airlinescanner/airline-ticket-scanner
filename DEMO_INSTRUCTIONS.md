# Инструкция по запуску демонстрационного приложения

## Быстрый старт

### 1. Установка зависимостей (если еще не установлены)

```bash
cd airline-ticket-scanner
npm install
```

### 2. Запуск приложения

```bash
npm start
```

После запуска вы увидите QR-код в терминале.

### 3. Просмотр на устройстве

**Вариант A: Реальное устройство (iOS/Android)**
1. Установите приложение **Expo Go** из App Store или Google Play
2. Отсканируйте QR-код из терминала
3. Приложение откроется в Expo Go

**Вариант B: iOS Simulator (только macOS)**
```bash
# Нажмите 'i' в терминале после npm start
# Или запустите напрямую:
npm run ios
```

**Вариант C: Android Emulator**
```bash
# Нажмите 'a' в терминале после npm start
# Или запустите напрямую:
npm run android
```

## Что вы увидите

Демонстрационное приложение показывает работу дизайн-системы:

### 📋 Секция "Цветовые токены"
- Визуализация основных цветов текущей темы
- background.app, background.card, accent.primary, button.primary

### 🔘 Секция "Компоненты кнопок"
- Primary Button (синяя кнопка-таблетка)
- Secondary Button (серая кнопка-таблетка)
- Disabled Button (неактивная кнопка)

### ✈️ Секция "Empty State"
- Компонент для отображения пустых состояний
- Используется в истории билетов и поиске авиакомпаний

### 📏 Секция "Токены отступов"
- spacing.horizontal: 16px
- spacing.vertical: 12px
- borderRadius.card: 12px

### 🎨 Кнопка переключения темы
- Нажмите для переключения между светлой и темной темой
- Анимация плавного перехода 300ms
- Тема сохраняется в AsyncStorage

## Проверка функциональности

### ✅ Переключение темы
1. Нажмите кнопку "Переключить на тёмную тему"
2. Наблюдайте плавную анимацию перехода (300ms)
3. Все компоненты должны обновиться одновременно

### ✅ Сохранение темы
1. Переключите тему на тёмную
2. Закройте приложение (полностью)
3. Откройте приложение снова
4. Тёмная тема должна сохраниться

### ✅ Системная тема (автоматическая)
1. В коде измените `setMode('system')` вместо `setMode('dark')`
2. Перезапустите приложение
3. Тема будет следовать системной теме устройства

## Возможные проблемы

### Проблема: "Metro bundler не запускается"
**Решение:**
```bash
# Очистите кеш Metro
npx expo start -c
```

### Проблема: "Expo Go не подключается"
**Решение:**
1. Убедитесь, что телефон и компьютер в одной Wi-Fi сети
2. Попробуйте подключиться через туннель:
```bash
npx expo start --tunnel
```

### Проблема: "TypeScript ошибки"
**Решение:**
```bash
# Переустановите зависимости
rm -rf node_modules package-lock.json
npm install
```
