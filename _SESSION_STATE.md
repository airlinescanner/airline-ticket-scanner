7. **Подготовлен установочный Release APK для телефона**:

   - Получен файл: `android/app/build/outputs/apk/release/app-release.apk` (~77 MB).
8. **Убрана dev-launcher оболочка из APK**:
   - Удалена зависимость `expo-dev-client` из `package.json`.
   - Выполнена чистая пересборка `./gradlew clean assembleRelease`.
   - Новый APK не содержит `Development Build` лаунчер.
9. **Улучшена стабильность и диагностика сканера**:

   - Добавлено локализованное сообщение `errorCameraClosed`.
