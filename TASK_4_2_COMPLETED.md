## Технічні деталі

### Архітектура
```
useLanguage (хук)
    ↓
useTranslation (react-i18next)
    ↓
i18n.changeLanguage()
    ↓
saveLanguage() → AsyncStorage
```

### API хука useLanguage

```typescript
const { currentLanguage, changeLanguage } = useLanguage();

// currentLanguage: 'uk' | 'ru'
// changeLanguage: (language: 'uk' | 'ru') => Promise<void>
```

### Приклад використання

```typescript
import { useLanguage } from './src/hooks/useLanguage';

function MyComponent() {
  const { currentLanguage, changeLanguage } = useLanguage();
  
  return (
    <Button 
      onPress={() => changeLanguage(currentLanguage === 'uk' ? 'ru' : 'uk')}
      title={`Мова: ${currentLanguage}`}
    />
  );
}
```
