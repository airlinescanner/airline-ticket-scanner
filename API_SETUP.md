# 🔑 Налаштування API ключів

Для роботи функції **"Оновити правила реєстрації"** потрібні API ключі від трьох сервісів.

## 📋 Необхідні API ключі

### 1. **Groq API** (безкоштовно)
Використовується для розпізнавання квитків через AI.

**Як отримати:**
1. Перейди на https://console.groq.com/keys
2. Зареєструйся або увійди
3. Натисни **"Create API Key"**
4. Скопіюй ключ (він виглядає як `gsk_...`)

**Ліміти безкоштовного плану:**
- 30 запитів/хвилину
- 14,400 запитів/день

---

### 2. **Mistral AI API** (безкоштовно)
Використовується для аналізу правил реєстрації авіакомпаній.

**Як отримати:**
1. Перейди на https://console.mistral.ai/api-keys/
2. Зареєструйся або увійди
3. Натисни **"Create new key"**
4. Скопіюй ключ

**Ліміти безкоштовного плану:**
- 5€ безкоштовних кредитів при реєстрації
- Модель `mistral-small-latest` коштує ~$0.001 за запит

---

### 3. **Tavily API** (безкоштовно)
Використовується для пошуку інформації про авіакомпанії в інтернеті.

**Як отримати:**
1. Перейди на https://app.tavily.com/home
2. Зареєструйся через Google або email
3. Після реєстрації ключ відразу з'явиться на головній сторінці
4. Скопіюй ключ (він виглядає як `tvly-...`)

**Ліміти безкоштовного плану:**
- 1,000 запитів/місяць

---

## ⚙️ Встановлення ключів

1. **Відкрий файл `.env`** в корені проекту `airline-ticket-scanner/`
2. **Вставте свої ключі:**

```env
EXPO_PUBLIC_GROQ_API_KEY=gsk_ваш_ключ_тут
EXPO_PUBLIC_MISTRAL_API_KEY=ваш_ключ_тут
EXPO_PUBLIC_TAVILY_API_KEY=tvly-ваш_ключ_тут
```

3. **Перезапусти Metro bundler:**
   - Зупини поточний процес (Ctrl+C в терміналі)
   - Запусти знову: `npx expo start --dev-client`

4. **Перезібери додаток в Xcode** (натисни ▶️)

---

## ✅ Перевірка

Після налаштування:
1. Відкрий додаток на iPhone
2. Перейди в **Налаштування**
3. Натисни **"Оновити правила реєстрації"**
4. Якщо все налаштовано правильно — побачиш прогрес оновлення

---

## 🔒 Безпека

- ❌ **НЕ комітьте файл `.env` в git!** (він вже в `.gitignore`)
- ✅ Використовуйте `.env.example` як шаблон для команди
- ✅ Зберігайте ключі в безпечному місці (менеджер паролів)

---

## 🆘 Проблеми?

**Помилка: "Failed to update airline database"**
- Перевір, чи правильно вставлені ключі в `.env`
- Перевір, чи перезапустив Metro bundler після зміни `.env`
- Перевір інтернет-з'єднання

**Помилка: "API Key is missing"**
- Переконайся, що ключі починаються з `EXPO_PUBLIC_`
- Перезапусти Metro bundler

**Помилка: "Rate limit exceeded"**
- Ти перевищив ліміт безкоштовного плану
- Зачекай кілька хвилин або оновлюй план
