## Что реализовано

### 1. ThemeContext (`src/theme/ThemeContext.tsx`)

Полнофункциональный React Context для управления темой оформления приложения:

- ✅ **Три режима темы**: `light`, `dark`, `system`
- ✅ **Подписка на системную тему**: Использует `Appearance.addChangeListener` для автоматического отслеживания изменений системной темы устройства
- ✅ **Персистентность**: Интеграция с `AsyncStorage` для сохранения и восстановления выбранной темы
- ✅ **Немедленное применение**: Изменения темы применяются мгновенно без перезапуска приложения
- ✅ **Типобезопасность**: Полная поддержка TypeScript

### 2. Хук useTheme

Удобный хук для доступа к контексту темы:

```tsx
const { mode, resolvedTheme, tokens, setMode } = useTheme();
```

- `mode` — текущий режим темы (light/dark/system)
- `resolvedTheme` — разрешенная тема (light или dark)
- `tokens` — токены текущей темы из `tokens.ts`
- `setMode` — функция для изменения режима темы

## Технические детали

### Зависимости

- `@react-native-async-storage/async-storage` — для персистентности
- `react-native` (Appearance API) — для подписки на системную тему

### Архитектурные решения

1. **Разделение ответственности**: 
   - `tokens.ts` — только данные (цвета, отступы)
   - `ThemeContext.tsx` — логика управления темой
   - `theme.ts` — типы TypeScript

2. **Оптимизация производительности**:
   - Использование `useCallback` для мемоизации функций
   - Минимальное количество ре-рендеров через правильное управление состоянием

3. **Обработка граничных случаев**:
   - Светлая тема по умолчанию при ошибке загрузки из AsyncStorage
   - Корректная обработка `null` от `Appearance.getColorScheme()`
   - Автоматическая отписка от слушателя системной темы при размонтировании

## Использование

```tsx
// В корневом компоненте App.tsx
import { ThemeProvider } from './src/theme/ThemeContext';

export default function App() {
  return (
    <ThemeProvider>
      <YourApp />
    </ThemeProvider>
  );
}

// В любом компоненте
import { useTheme } from './src/theme/ThemeContext';

function MyScreen() {
  const { tokens, setMode } = useTheme();
  
  return (
    <View style={{ backgroundColor: tokens.colors.background.app }}>
      <Text style={{ color: tokens.colors.text.primary }}>
        Привет, мир!
      </Text>
    </View>
  );
}
```
